# jira-cli

A CLI tool that reads a Jira issue — including its **description**, **comments**, and all **attached documents** — and outputs a single clean structured Markdown document.

Supports attachment types: `.txt`, `.md`, `.csv`, `.json`, `.yaml`, `.log`, `.pdf`, `.docx`, `.xlsx`.

## Quick Start

```bash
# 1. Set up the virtual environment (one-time)
cd services/jira-cli
python -m venv .venv
source .venv/bin/activate        # macOS/Linux
pip install -r requirements.txt

# 2. Configure Jira credentials (one-time)
cp .env.example .env             # then edit .env

# 3. Run
python jira_cli.py PROJECT-123
```

## Configuration

Copy `.env.example` to `.env` and fill in your Jira credentials:

```bash
cp .env.example .env
```

| Variable | Description |
|---|---|
| `JIRA_URL` | Your Jira base URL, e.g. `https://yourorg.atlassian.net` |
| `JIRA_USER` | Your Jira account email |
| `JIRA_API_TOKEN` | API token from [id.atlassian.com](https://id.atlassian.com/manage-profile/security/api-tokens) |

## Usage

```
python jira_cli.py <ISSUE-KEY> [options]
```

### Arguments

| Flag | Description |
|---|---|
| `issue_key` | **(required)** Jira issue key, e.g. `PROJECT-123` |
| `-o`, `--output FILE` | Write output to a file instead of stdout |
| `--no-attachments` | Skip downloading attachments (list filenames only) |
| `--comments-limit N` | Show only the last N comments (default: show all) |
| `--env-file PATH` | Path to `.env` file (default: `.env`) |

## Examples

**Print to terminal:**
```bash
python jira_cli.py PROJECT-123
```

**Save to a Markdown file:**
```bash
python jira_cli.py PROJECT-123 --output issue.md
```

**Skip attachment download:**
```bash
python jira_cli.py PROJECT-123 --no-attachments
```

**Show only last 5 comments:**
```bash
python jira_cli.py PROJECT-123 --comments-limit 5
```

**Use a custom env file:**
```bash
python jira_cli.py PROJECT-123 --env-file /path/to/.env
```

```

|-------------|--------------|
| Status      | In Progress  |
| Priority    | High         |
| Assignee    | Jane Doe     |
| ...         | ...          |

## Description
...

## Attachments (N)
### filename.pdf
- Size: 12,345 bytes
- Uploaded by: John Smith
- Content:
  ...extracted text...

## Comments (N)
### Comment 1 — Jane Doe on 2025-01-15 10:30 UTC
...

========================================================================
*Generated by jira-cli on 2025-01-15 10:35 UTC*
```
